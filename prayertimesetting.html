Prayer Times Table Schema

The following SQL statement creates a table for storing prayer times for different islands and dates: 
CREATE TABLE prayer_times (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    island VARCHAR(50),
    date DATE,
    imsaak TIME,
    fajr TIME,
    sunrise TIME,
    dhuhr TIME,
    asr TIME,
    maghrib TIME,
    isha TIME
);
